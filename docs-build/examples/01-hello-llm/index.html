    🎉 첫 AI 대화 시작하기¶

    드디어 AI와 대화할 시간입니다! 이 섹션에서는 가장 기본적인 것부터 차근차근 배워봅시다.

    • ✅ AI가 무엇인지 쉽게 이해하기
    • ✅ 첫 번째 AI 프로그램 만들기
    • ✅ AI의 응답 이해하기
    • ✅ 다양한 질문 해보기

    • 토큰과 비용 이해하기
    • 좋은 질문하는 방법

    📝 이 섹션의 목표¶

    이 섹션을 끝내면 다음과 같은 코드를 이해하고 작성할 수 있습니다:

    # AI와 대화하는 프로그램
from pyhub.llm import LLM

# AI 도우미 만들기
ai_helper = LLM.create("gpt-4o-mini")

# 질문하고 답변 받기
answer = ai_helper.ask("파이썬은 왜 이름이 파이썬인가요?")
print(answer.text)
