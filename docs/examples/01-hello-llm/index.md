## 📝 이 섹션의 목표

이 섹션을 끝내면 다음과 같은 코드를 이해하고 작성할 수 있습니다:

```python
# AI와 대화하는 프로그램
from pyhub.llm import LLM

# AI 도우미 만들기
ai_helper = LLM.create("gpt-4o-mini")

# 질문하고 답변 받기
answer = ai_helper.ask("파이썬은 왜 이름이 파이썬인가요?")
print(answer.text)
```
